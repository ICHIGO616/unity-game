# unity-game

Unityで制作した3Dのホラーゲームです。

## 概要

村を舞台にした一人称視点の3Dホラーゲームです。
プレイヤーは村を歩きながら、NPCや環境のインタラクティブ要素と関わったり、
敵から逃げながらゲームを進めます。

## 使用技術

- Unity 6.3 LTS（HDRP）
- C#
- Meshy.AI

## 動作環境

- Windows PC

## 実装機能

- 敵AIの追跡・攻撃行動
- プレイヤーのHP管理・ダメージ処理・死亡処理
- NPCとのダイアログシステム
- BGMクロスフェード・足音などのサウンド管理
- シーン遷移（タイトル・ゲーム・ゲームオーバー）

## 制作人数

個人制作（1人）
